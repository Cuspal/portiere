# Getting Help

- **Bug reports / feature requests:** [GitHub Issues](https://github.com/Cuspal/portiere/issues) — please use the templates.
- **Security reports:** see [SECURITY.md](SECURITY.md) — do **not** open a public issue.
- **Questions / discussion:** [GitHub Discussions](https://github.com/Cuspal/portiere/discussions).
- **Documentation:** [docs/documentations/](docs/documentations/), Jupyter notebooks under [docs/notebooks_examples/](docs/notebooks_examples/).
- **Commercial / cloud-hosted offering:** https://portiere.io.

If you are reporting a problem, please include:

- Portiere version (`python -c "import portiere; print(portiere.__version__)"`)
- Python version, OS
- Minimal reproducer (paste a snippet, not a screenshot)
- Full stack trace if applicable
- For data quality / mapping issues: a tiny synthetic example that reproduces (do **not** include PHI)

For commercial support, custom integrations, or hosted offerings, contact us at hello@cuspal.co.
